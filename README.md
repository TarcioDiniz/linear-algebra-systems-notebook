# Sistemas Lineares: Teoria, Geometria e Aplicações com Python

Este repositório contém uma série de notebooks Jupyter dedicados ao estudo de sistemas lineares, combinando conceitos teóricos, visualização geométrica e implementação prática usando Python.

## 🚀 Como Começar

Para uma visão geral do projeto, motivação e o roteiro completo de estudos, acesse o notebook de entrada:

👉 **[Notebook 00: Overview e Roteiro de Estudo](./notebooks/00_overview_roteiro.ipynb)**

## 📚 Conteúdo da Série

1. **[01] Introdução:** Definições e Geometria 2D.
2. **[02] Matriz Aumentada:** Organização e Representação.
3. **[03] Escalonamento:** Algoritmo de Gauss-Jordan.
4. **[04] Geometria 3D e Dependência Linear:** Vetores LI/LD e Posto.
5. **[05] Transformações Lineares:** Núcleo, Imagem e Invertibilidade.
6. **[06] Aplicação: Fluxo de Tráfego:** Modelagem de redes urbanas.
7. **[07] Aplicação: Interpolação Polinomial:** Ajuste de curvas.
8. **[08] Exercícios Resolvidos:** Prática Manual vs Computacional.
9. **[09] Conclusão:** Síntese e Próximos Passos.

## 🛠️ Requisitos

O projeto utiliza `uv` para gerenciamento de dependências. Para instalar:

```bash
uv sync
```

Principais bibliotecas: `numpy`, `sympy`, `matplotlib`, `seaborn`, `networkx`.
